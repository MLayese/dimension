Sound Design

Sound Design Section: Redesigns, Foley, Implemented SFX, and technical SFX

    • Animalese

    This Unity-based implementation demonstrates how Animalese-style speech can be recreated using Wwise and real-time audio parameter control. Each character (letter) is assigned it's own unique wav file that modulates in both pitch and speed. Special characters such as !* or ? are not assigned a wav file.

    Features include:

    • Dynamic pitch modulation
    • C# scripting and Wwise integration

    • Spider Man SFX Redesign

    A redesign of one of the Gameplay Trailers! Original audio was stripped from the video. Audio was rebuilt from the ground up using Ableton Live and MaxMSP for sound manipulation.

    Features include:

    • New sounds recorded via Zoom H4N and MaxMSP
    • Implemnted and Mixed in Ableton
