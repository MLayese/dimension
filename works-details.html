Sound Design

Sound Design Section: Redesigns, Foley, Implemented SFX, and technical SFX

    • Audio Reel

    Series of Audio clips produced by me

    Features include:

    • Techincal Implementation of Audio via Unity and Wwise
    • Sound Designs and Redesigns from various visual mediums
    • Recorded using Zoom H4N, Boom Library, and GDC 2023 Library, processed in Ableton 11

    • Animalese

    This Unity-based implementation demonstrates how Animalese-style speech can be recreated using Wwise and real-time audio parameter control. Each character (letter) is assigned it's own unique wav file that modulates in both pitch and speed. Special characters such as !* or ? are not assigned a wav file.

    Features include:

    • Dynamic pitch modulation
    • C# scripting and Wwise integration

    • MaxMSP

    Technical Demonstration of MSP Patch with seamless, randomized ambient audio. For this specific patch, a folder filled with ambient loops selects four random files to go through. Each file is assigned to a fourth of a "square", where a random variable dictates which sounds take priority, which will be in the background, etc. etc. Fade in and out functions are built in. A container with one shots for specific ambiances will trigger occasionally. Overall, a very nifty tool and fairly easy to set up and use. Public MaxMSP Patch will come soon.

    Features include:

    • Dynamic pitch and audio modulation
    • MaxMSP scripting
    • MaxForLive PlugIn

    • Spider Man SFX Redesign

    A redesign of one of the Gameplay Trailers! Original audio was stripped from the video. Audio was rebuilt from the ground up using Ableton Live and MaxMSP for sound manipulation.

    Features include:

    • New sounds recorded via Zoom H4N and MaxMSP
    • Implemnted and Mixed in Ableton

    • AR Sound designer

    Sound design for Furniture Tech Demo. Utilized UE 5 with Wwise for audio integration. Wanted to adhere to their design of functionality and minimalism so all sounds are based off of real furniture we had access to during the project. Noises were recorded using a Zoom H4N and a Sennheiser 600 shotgun. For processing, MaxMSP acted as the medium

    Features include:

    • Sounds recorded via Zoom H4N
    • Processed in MaxMSP
    • Implemented in Wwise

    • Processed in Ableton
    • Implemented in UE Blueprints

    • Sound Design and Dialogue

    Ohhh I loved this one! I stripped the original audio (free Unity Project btw) and placed my own. The dialogue heard was added in later; I wanted to add more sci-fi flair and this was the result! Most sounds were recorded in a proper foley pit with the exception being the room tone being made in Ableton's wavetable.

    Features include:

    • Sounds recorded via Zoom H4N, MKE 600, SM57
    • Sounds synthesized and Processed and in Ableton and MaxMSP
